## FA♂Q
- gitment登陆初始化出现object ProgressEvent
  原因是作者的服务器网站证书过期了，手动打开以下链接，出现`您的连接不是私密连接`，点高级继续前往即可`继续前往gh-oauth.imsun.net（不安全）`，这时回到博客页面刷新即可初始化评论
  https://gh-oauth.imsun.net
